# COMANDOS DE git
# git show = muestra los cambios hechos de los commit realizados
# git diff numerodelcommit = mustra los cambios realizados de ese commit a la version nueva
sin haber hecho el commit actual
#  git diff numerodelcommit numerodelcommitAcomparar= mustra las diferecias realizadas
entre los dos commit.
# git status muestras los archivos modificados en esa carpeta donde esta el repositorio.
# git log = muestra todos los commit ordenados del actual al primero realizado.
# git log --stat = muestra los cambios en detalle de los commit
# git checkout numerodelcommit = mustra la version de ese commit
# git checkout master = me trae la ultima version realizada con commit
# git push = comando para actualizar los cambios de mi repositorio local
a mi repositorio remoto.
# git fetch = actualiza mi repositorio local del repositorio remoto ya sea por que un
integrante del equipo haya actualizado el código.
# git merg = fusiona los cambios del repositorio remoto a local.
# git pull = actualiza mi repositorio local a mi repositorio remoto de un solo paso.
